# VS Code Keyboard Shortcuts for Windows

## Window / Editor Management
- New Window: `Ctrl+Shift+N`
- Close Window: `Ctrl+Shift+W`
- Split Editor: `Ctrl+\`
- Toggle Editor Group Layout: `Ctrl+Alt+1` / `Ctrl+Alt+2` / `Ctrl+Alt+3`
- Focus into Editor Group (1–9): `Ctrl+K Ctrl+<number>`
- Close Editor: `Ctrl+W`
- Close Folder: `Ctrl+K F`
- Close All Editors: `Ctrl+K W`

## File Management
- New File: `Ctrl+N`
- Open File: `Ctrl+O`
- Save: `Ctrl+S`
- Save As: `Ctrl+Shift+S`
- Save All: `Ctrl+K S`
- Revert File: `Ctrl+K R`
- Close File: `Ctrl+F4`

## View / Panels
- Show Explorer: `Ctrl+Shift+E`
- Show Search: `Ctrl+Shift+F`
- Show Source Control: `Ctrl+Shift+G`
- Show Run / Debug: `Ctrl+Shift+D`
- Show Extensions: `Ctrl+Shift+X`
- Show Terminal: `Ctrl+`
- Toggle Sidebar Visibility: `Ctrl+B`
- Toggle Panel: `Ctrl+J`
- Toggle Full Screen: `F11`
- Toggle Zen Mode: `Ctrl+K Z`

## Navigation
- Go to File: `Ctrl+P`
- Go to Symbol: `Ctrl+Shift+O`
- Go to Definition: `F12`
- Peek Definition: `Alt+F12`
- Go Back: `Alt+LeftArrow`
- Go Forward: `Alt+RightArrow`

## Search / Replace
- Find: `Ctrl+F`
- Replace: `Ctrl+H`
- Find in Files: `Ctrl+Shift+F`
- Replace in Files: `Ctrl+Shift+H`

## Editor Editing
- Cut Line: `Ctrl+X`
- Copy Line: `Ctrl+C`
- Move Line Up/Down: `Alt+Up` / `Alt+Down`
- Duplicate Line: `Shift+Alt+Down`
- Delete Line: `Ctrl+Shift+K`
- Insert Line Below: `Ctrl+Enter`
- Insert Line Above: `Ctrl+Shift+Enter`
- Comment Line: `Ctrl+/`
- Block Comment: `Shift+Alt+A`
- Format Document: `Shift+Alt+F`
- Toggle Word Wrap: `Alt+Z`

## Terminal
- New Terminal: `Ctrl+Shift+`
- Kill Terminal: `Ctrl+Shift+W`
- Split Terminal: `Ctrl+Shift+5`
- Focus Terminal: `Ctrl+`

## Command Palette
- Show All Commands: `Ctrl+Shift+P` or `F1`

## Useful Window Shortcuts
- Minimize Window: `Alt+Space` then `N`
- Maximize Window: `Alt+Space` then `X`
- Restore Window: `Alt+Space` then `R`
- Close Window: `Alt+F4`
